# Astro i18n Example

This is a demo project for `@ldesign/i18n-astro`.

## Usage

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Features

- Basic translation with `i18n.t()`
- Language switching via URL parameters
- Server-side rendering
- Counter interaction
