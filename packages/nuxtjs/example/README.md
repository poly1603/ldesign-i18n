# Nuxt.js i18n Example

This is a demo project for `@ldesign/i18n-nuxtjs`.

## Usage

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Generate static site
pnpm generate

# Preview production build
pnpm preview
```

## Features

- Basic translation with `t()`
- Language switching
- Server-side rendering support
- Counter interaction
