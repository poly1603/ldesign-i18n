# Next.js i18n Example

This is a demo project for `@ldesign/i18n-nextjs`.

## Usage

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Features

- Basic translation with `t()`
- Language switching
- Client-side rendering with App Router
- Counter interaction
