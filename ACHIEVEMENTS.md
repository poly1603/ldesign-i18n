## 🌟 技术亮点

### 1. 统一的 API 设计
所有框架适配包提供一致的 API：
```typescript
const { t, locale, setLocale } = useI18n()
```

### 2. 智能类型推导
```typescript
// 类型安全的键名
i18n.t('common.hello') // ✅
i18n.t('invalid.key')  // ❌ TypeScript 错误
```

### 3. 高性能缓存
```typescript
// LRU 缓存，O(1) 性能
const cached = i18n.t('frequently.used.key')
```

### 4. 灵活的插值
```typescript
// 支持复杂的参数插值
t('greeting', { name: 'World', time: new Date() })
```

### 5. 完整的 SSR 支持
```typescript
// Next.js、Nuxt、Remix、SvelteKit 等
// 都支持服务端渲染
```
